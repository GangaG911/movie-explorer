# 🎬 Movie Explorer

A modern movie search web application built using JavaScript that allows users to discover movies, view details, and manage favorites.

---

## 🚀 Features

* 🔍 Search movies in real-time
* 🎬 View movie details (title, year, rating, plot)
* ❤️ Add movies to favorites
* 🎥 Watch trailer (YouTube search)
* 📱 Responsive UI design

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* OMDb API

## ⚙️ How to Run

1. Download or clone the repository
2. Open project in Visual Studio Code
3. Replace API key in `app.js`:

```javascript

```

4. Open `index.html` in browser
